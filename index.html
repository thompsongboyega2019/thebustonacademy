Front-end Development

Front-end development is the practice of building the part of a website or application that users interact with directly. It's what users see and interact with when they visit a website or use an application. This course is designed to expose you to the various Front-end technologies, frameworks and libraries such as HTML, CSS, JavaScript, jQuery, Bootstrap etc. used in developing the UI/UX of a website or an application.

Back-end Development

Back-end development is the part of web development that is responsible for creating and maintaining the server-side of a website or application. In this course, you will learn how to work with programming languages like PHP, Python and Node.js as well as databases such as MySQL, PostgreSQL, and MongoDB. You will learn how to build the functionality that allows users to interact with the website or application.

Mobile Application Development

Mobile application development is the process of creating software applications that run on mobile devices, such as smartphones and tablets. In this course you will learn the various programming languages and frameworks used in developing mobile apps. These languages include: Swift(iOS app development), Java(Android app development), Ionic and React Native(Cross-platform app development).

Content & Digital Marketing

This course encompasses all marketing efforts that use digital channels to reach and engage with target audience. In this course, you will learn the strategic approach to creating and distributing valueable, relevant, and consistent content to attract and retain a clearly-defined audience. You will be taught various digital channels like websites, search engines, social media platforms, email, mobile apps, and more.

UI/UX Design

UI/UX design refers to the process of designing the user interface (UI) and user experience (UX) of a digital product, such as a website, mobile app, or software application. This course will teach you how to create a digital product that is visually appealing, easy to use, and provides a seamless user experience. Additionally, you will learn the various tools that are used to create stunning product designs.

Software Programming

It is the process of writing code to create computer software, which include applications, operating systems, and other software products. Software programming involves designing, coding, testing, and debugging software applications to ensure they function as intended. In this program, you will learn the various programming languages used in software programming such as: Java, Python, JavaScript, PHP e.t.c
